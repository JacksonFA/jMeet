## Para executar o projeto

### Backend

- Use node >= v18.17
- `cd backend`
- `yarn`
- `yarn dev`
- access <http://localhost:3333>

### Frontend

- Use node >= v18.17
- `cd frontend`
- `yarn`
- create a .env file to set backend url
  - e.g.: `NEXT_PUBLIC_API_URL='http://localhost:3333'`
- `yarn dev`
- access <http://localhost:3000>
